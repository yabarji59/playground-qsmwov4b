Playground JavaScript

1. Variables et Types de données

Déclarez et utilisez différentes variables en JavaScript.

let nombre = 42;
let texte = "Hello World";
let booleen = true;
let tableau = [1, 2, 3];
let objet = { nom: "John", age: 30 };

2. Tableaux

Manipulez des tableaux avec différentes méthodes.

let fruits = ['pomme', 'banane', 'orange'];
fruits.push('fraise');
fruits.unshift('kiwi');
fruits.pop();
fruits.shift();

3. Conditions

Utilisez les structures conditionnelles.

if (age >= 18) {
    console.log("Majeur");
} else {
    console.log("Mineur");
}

4. Boucles

Explorez les différentes boucles en JavaScript.

for (let i = 1; i <= 5; i++) {
    console.log(i);
}

while (compteur <= 3) {
    console.log(compteur);
    compteur++;
}

5. Fonctions

Créez et utilisez des fonctions.

function addition(a, b) {
    return a + b;
}

const multiplication = (a, b) => a * b;

function saluer(nom = "visiteur") {
    return `Bonjour ${nom} !`;
}
